# 文章编辑页面优化设计

## 概述

优化博客管理后台的文章写作页面，参考掘金/CSDN 的写作体验，采用极简专注模式，让用户更加专注于写作本身。

**设计日期：** 2026-04-06

## 设计决策

| 决策项 | 选择 |
|--------|------|
| 布局方案 | 极简专注模式（标题顶栏 + 全宽编辑器） |
| 设置面板交互 | 弹出模态框（发布前设置封面、分类、标签、摘要） |
| 预览模式 | 可切换预览（点击按钮切换编辑/预览视图） |

## 页面布局设计

### 1. 顶部导航栏（height: 60px）

```
┌────────────────────────────────────────────────────────────┐
│ ← 写文章                    [预览] [草稿] [发布]  自动保存 · 3分钟前 │
└────────────────────────────────────────────────────────────┘
```

- 左侧：返回按钮（ArrowLeft 图标） + 页面标题
- 右侧操作按钮：
  - **预览**：切换预览模式
  - **草稿**：保存草稿（status: 0）
  - **发布**：弹出设置模态框，确认后发布（status: 1）
- 显示自动保存状态提示

### 2. 标题区域

```
┌────────────────────────────────────────────────────────────┐
│  文章标题输入框（font-size: 24px, font-weight: 600）           │
```

- 大号标题输入框，无边框设计，底部细线分隔
- 实时显示字数统计
- 已选分类和标签以小标签形式显示在标题下方
- 点击标签可快速打开设置模态框修改

## 响应式适配

### 桌面端（≥1024px）
- 编辑器全宽显示
- 工具栏按钮完整显示

### 平板端（768px - 1023px）
- 工具栏按钮适当压缩
- 模态框宽度调整为 90%

### 移动端（<768px）
- 标题输入框字号减小（18px）
- 工具栏隐藏次要按钮，使用下拉菜单
- 模态框全屏显示

## 技术实现要点

### 关键交互

1. **发布流程**：
   - 点击"发布"按钮 → 弹出设置模态框
   - 填写/确认封面、分类、标签、摘要
   - 点击"保存并发布" → 调用 createArticle/updateArticle API

2. **快速修改分类/标签**：
   - 点击标题区的分类/标签标签 → 弹出设置模态框
   - 修改后点击"保存设置"返回编辑

3. **预览切换**：
   - 点击"预览"按钮 → showPreview = true → 显示渲染内容
   - 点击"返回编辑" → showPreview = false → 显示编辑器
